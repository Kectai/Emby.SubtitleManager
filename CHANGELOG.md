## v1.3.2 - 2026-05-08

- 精简公开文档，将安装、编译、使用、API 和项目结构说明合并到根目录 README。
- 删除重复的 docs 说明文件，避免多处文档内容不一致。
- 更新插件列表图标，移除多余外层卡片轮廓并优化主体比例和留白。
- 同步本地构建脚本中的图标校验和插件目录提示。
- 添加 MIT License，并同步项目包元数据和 README 许可证说明。
- 后端接口显式限制为 Emby 管理员账号访问。

## v1.3.1 - 2026-05-08

- 更新 README，使功能、安全限制、构建、安装目录和插件图标说明与当前实现保持一致。
- 替换插件图标为新的矩形字幕管理缩略图，用于 Emby“高级 - 插件”页面展示。
- 将左侧主菜单入口图标调整为 Emby 内置 `subtitles` 图标。
- 移除公开仓库中的图标生成脚本，只保留最终图标资源。

## v1.3.0 - 2026-05-08

- 加固字幕删除接口：只允许删除当前媒体项已识别的外部字幕，且路径必须位于该媒体的 Emby 元数据目录内。
- 加固字幕上传接口：后端限制字幕格式、语言代码、空文件和 20MB 文件大小，并拒绝覆盖同名字幕。
- 将删除后的元数据刷新改为异步 `await`，避免同步阻塞。
- 将 extras 虚拟文件夹 ID 从 `string.GetHashCode()` 改为基于 SHA256 的稳定 ID，并为 extras 缓存增加过期清理。
- 优化媒体列表性能：列表接口默认不再为每个媒体项附带字幕流，前端改为选中媒体时按需加载字幕信息。
- 前端上传前增加文件大小、空文件、格式白名单和扩展名一致性校验。
